# Sirah Digital

This project is the official website for Sirah Digital, an AI automation agency for healthcare and businesses.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Getting Started

To run the project locally:

1.  Clone the repository
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`

The frontend will be available at `http://localhost:8081` and the backend at `http://localhost:5000`.
